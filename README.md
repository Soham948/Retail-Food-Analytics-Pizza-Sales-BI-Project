# Retail-Food-Analytics-Pizza-Sales-BI-Project
This project delivers a complete end-to-end data analytics solution for a pizza retail business using SQL, Python, and Power BI. It focuses on extracting actionable insights from transactional sales data, including revenue trends, customer ordering behavior, product performance, and category-wise sales distribution.


# 🍕 Pizza Sales Analytics & BI Dashboard

## 📌 Project Overview
This project focuses on analyzing pizza sales data to extract meaningful business insights using SQL, Python, and Power BI. The goal is to understand sales performance, customer behavior, and product trends to support better decision-making.

---

## 🎯 Objectives
- Analyze total revenue and sales performance
- Identify top-selling and low-performing pizzas
- Understand customer ordering patterns
- Track monthly and daily sales trends
- Visualize insights using an interactive dashboard

---

## 🛠️ Tech Stack
- **SQL** – Data querying and analysis
- **Python (Jupyter Notebook)** – Data preprocessing & exploration
- **Power BI** – Dashboard and data visualization
- **CSV Dataset** – Raw data source

---

## 📂 Project Structure
📁 Pizza-Sales-Analytics
│── 📄 pizza_sales.csv # Raw dataset

## 🚀 How to Use

1. Load the dataset (`pizza_sales.csv`) into your SQL database
2. Run queries from `pizzas.sql`
3. Open `Pizza_connection.ipynb` for Python-based analysis
4. Open `pizza_dashboard.pbix` in Power BI to view the dashboard
